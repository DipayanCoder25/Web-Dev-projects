Random Number Printer

This project demonstrates printing a set of random numbers using a while loop with the let keyword in JavaScript. Adjust the parameters below and generate random numbers!

How It Works

This program uses a while loop to generate a specified number of random values. The let keyword is used to declare the loop counter variable, which is block-scoped to the loop.

Each iteration generates a random number within the specified range and displays it visually on the screen. You can control how many numbers to generate and their minimum/maximum values.

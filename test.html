Array Mapper & Sorter - Demo

Demonstrating array operations without user input

Demo Configuration

How This Demo Works:

This demo uses a predefined array: [12, 5, 8, 21, 3, 17, 9, 14]

Click on any operation button to apply that mapping operation to the array.

The results will show the original array, mapped array, and both ascending and descending sorted versions.

Current Operation: Double (multiply each element by 2)
JavaScript Code Preview:
// Original array
const originalArray = [12, 5, 8, 21, 3, 17, 9, 14];
 
// Map the array (example with double operation)
const mappedArray = originalArray.map(num => num * 2);
 
// Sort in ascending order
const ascendingArray = [...mappedArray].sort((a, b) => a - b);
 
// Sort in descending order
const descendingArray = [...mappedArray].sort((a, b) => b - a);
